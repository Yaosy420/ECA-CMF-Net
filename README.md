# ECA-CMF Net

This repository is associated with the paper **“ECA-CMF Net: An ECA-Enhanced Conditional Modulation Fusion Network for Multimodal Sentiment Analysis”**, accepted by *Electronics*.

## Paper Information

**Title:** ECA-CMF Net: An ECA-Enhanced Conditional Modulation Fusion Network for Multimodal Sentiment Analysis
**Authors:** Shengyuan Yao, Shuxia Ren
**Journal:** Electronics
**Manuscript ID:** electronics-4324900

## Overview

ECA-CMF Net is a multimodal sentiment analysis framework designed for sentiment classification on benchmark datasets such as CMU-MOSI and CMU-MOSEI. The model integrates Efficient Channel Attention (ECA) and Conditional Modulation Fusion (CMF) to enhance modality-specific representation learning and adaptive multimodal fusion.

## Code Availability

The source code and experimental configuration files are currently being organized and will be made available in this repository.

The original experimental scripts contain local paths and environment-specific settings. We are currently cleaning and documenting the code to improve readability and reproducibility before public release.

Additional implementation details are available from the corresponding author upon reasonable request.

## Datasets

The experiments are conducted on the publicly available CMU-MOSI and CMU-MOSEI datasets. Due to dataset licensing and distribution restrictions, the original datasets are not included in this repository.

Please obtain the datasets from their official sources and follow the corresponding dataset usage policies.

## Reproducibility

The experimental results may vary slightly due to random initialization, hardware differences, CUDA versions, and dataset preprocessing settings. The main implementation files, configuration files, and usage instructions will be updated in this repository.
